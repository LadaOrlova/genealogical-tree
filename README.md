# Генеалогическое древо

Интерактивная карта семейного архива Орловых — Ткачёвых — Белановых — Фёдоровых — Самусёвых — Воробьёвых.

## Что внутри

- **Интерактивный граф** — Cytoscape.js + dagre-layout. Зум колесом, пан перетаскиванием, клик по человеку открывает карточку со всеми известными деталями.
- **Шесть линий рода**, каждая со своим цветом и фильтром в боковой панели.
- **Маркер неуверенности** — пунктирная рамка и плашка в карточке, если имя или родственная связь пока не подтверждены.
- **Поиск** по имени, профессии и месту.
- **Открытые вопросы** — храним прямо в карточках людей, чтобы при следующем разговоре с бабушкой было куда смотреть.

## Как открыть локально

Достаточно открыть `index.html` в браузере. Это обычный статичный сайт без сборки.

## Как добавить нового человека

1. Открыть [`data.js`](./data.js).
2. Добавить объект в массив `people`. Минимум: `id`, `name`, `fullName`, `line`, `gen`, `certain`.
3. Если у человека есть родители или супруг — добавить/расширить запись в массиве `marriages`.

`line` — одно из:

| Ключ | Линия |
| --- | --- |
| `you` | Владлена |
| `maternal-tkachev` | Ткачёвы · Белановы (по маме) |
| `maternal-fedorov` | Фёдоровы (по маме) |
| `maternal-samusev` | Самусёвы (по маме) |
| `paternal-vorobyov` | Воробьёвы (по папе) |
| `orlov` | Орловы (по папе) |
| `bridge` | Связующие |

`gen` — поколение относительно Владлены, `0` это её уровень, чем больше цифра — тем глубже в предки.
